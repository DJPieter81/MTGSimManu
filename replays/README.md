# MTG Bo3 Replay Logs
Raw verbose output from `run_meta.py --bo3`. Each file is a complete match log parseable by the replay HTML builder.

## Naming convention
`{deck1}_{deck2}_s{seed}.txt` — all lowercase, spaces → underscores.
